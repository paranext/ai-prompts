# ai-prompts
Place for team members to save and share AI prompt files they find useful in their own work

Please create a top-level directory for yourself and store whatever AI prompt file you find useful. Structure your directory and files however you like. No PRs required.

Please be respectful and don't modify anyone else's AI prompt files. However, please copy and reuse anything you like from anyone else. This is meant to be a place to help us learn and share.
